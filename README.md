Event Scheduling App is basic app in which to schedule some task in the calendar .
User can select start date and end date , user can select frequency , interval and add  task description in it.
The task will show on basis of frequency like daily, weekly, monthly, yearly, custom days task , then  it will show the task on calender on specific date ranges .
